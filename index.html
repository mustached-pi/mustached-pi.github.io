    The Mustached Pi Project

    The MPi Project is aimed to provide a low-cost and open source home automation solution based on the Raspberry Pi mini computer.

    You can try the public ONLINE BETA INSTALLATION.

    As it is not finished, the hardware scheme is not publicy available yet, but you can find the software both of the client app and the server app.

    mustached-pi-client

    Mostly written in python, it runs on the Raspberry Pi and it tells the server about sensors' values and outputs what the server tells it to.

    Learn more visiting the project page of mustached-pi-client.

    mustached-pi-server

    Mostly written in PHP5 (PSR-0 compliant), it runs on a web server and uses a MySQL database to store and analyze Client's data.

    Learn more visiting the project page of mustached-pi-server.
